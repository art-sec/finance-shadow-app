# Site Financas

App React Native (Expo) para painel financeiro com login Firebase e cadastro de usuarios.

## Como rodar

1. `npm install`
2. `npm run web` para web ou `npm run android`/`npm run ios`

## Firebase

Atualize as credenciais em src/firebase/config.ts com os dados do seu projeto Firebase.
Ative o login por email e senha no Firebase Auth.
Habilite o Firestore para armazenar perfis em `users/{uid}`.
